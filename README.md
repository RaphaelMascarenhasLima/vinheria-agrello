# vinheria-agnello
O projeto Vinheria Agnello foi desenvolvido com o objetivo de criar um site institucional e de e-commerce para uma vinheria localizada em São Paulo, que atua há mais de 15 anos no mercado de vinhos nacionais e internacionais.
A proposta do projeto é levar a experiência da loja física para o ambiente digital, oferecendo informações sobre vinhos, tipos de uvas, harmonizações e um atendimento mais próximo do cliente por meio do site.

O sistema foi desenvolvido utilizando HTML e CSS, seguindo a estrutura semântica e os requisitos solicitados pela disciplina.O site é composto pelas seguintes páginas:
index.html
Página inicial do site, contendo apresentação da vinheria, menu de navegação e informações gerais sobre a empresa.

vinhos.html
Página que apresenta os tipos de vinhos disponíveis, com imagens, lista de categorias e tabela de vinhos.

mundodosvinhos.html
Página que explica o mundo dos vinhos, variedades de uvas, armazenamento e harmonização.

sobre.html
Página com a história da vinheria, informações da equipe e formulário de contato.

style.css
Arquivo responsável pela estilização do site.

efeitos.css
Efeitos Visuais Aplicados
Pseudo-classes utilizadas

Foram aplicadas pseudo-classes para melhorar a interatividade e experiência do usuário:

:hover → Utilizado em botões e links do menu para alterar cores, ampliar elementos e criar respostas visuais ao passar o mouse.
:focus → Aplicado em campos de formulário para destacar bordas e facilitar a usabilidade durante o preenchimento.
:nth-child(even) → Usado nas tabelas de vinhos para alternar cores de linhas e melhorar a leitura visual.
Pseudo-elementos utilizados

Elementos decorativos foram adicionados sem alterar a estrutura HTML:

::before → Inserção de ícones decorativos (🍷) antes de títulos principais.
::after → Criação de linhas decorativas abaixo de subtítulos para reforçar identidade visual.
Animações aplicadas

O projeto utiliza animações para modernizar a navegação:

@keyframes fadeBanner → Efeito de surgimento suave no banner principal da página inicial.
Transições (transition) → Aplicadas em links, botões e formulários para suavizar mudanças de cor, tamanho e interação.
